(About the author)
 
     This is a handy module which allows you to create an object from an Active-X server (OCX, DLL) without the need to register it first. >> Updated: removed Prog ID check because it required the object to be registered... changed Prog ID check to Class ID check, so you'll need the class name now. To get it, use a type library viewer of your choice or my type library browser submission.
